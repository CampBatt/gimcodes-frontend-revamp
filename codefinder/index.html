    How It Works

    Gimkit games need a code to be able to join them, this tool uses a brute force method to find working codes.

    When you enter a code to join a Gimkit game your browser sends a request to the Gimkit API to get info about the game that you're joining. If you give an incorrect code you the API responds with an error, if you supply a valid code the API will give the info. The code finder abuses this to find codes that work.
